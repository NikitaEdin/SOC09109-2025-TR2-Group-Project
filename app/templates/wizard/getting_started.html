Get Started

Different Types of Flights

Urban

Projects siutated in developed, built-up areas where police must be informed before flying, and a leaflet drop is often required.

Rural

Projects located in open or remote areas where wildlife restrictions apply, and a loading list is typically necessary.

Project Creation Process

1) Initiate Project: Log in, navigate to the dashboard, and click the green "New Project" button.
2) Operation Location: Select the location on the map by clicking on the desired point.
3) Project Type: Select either "Rural" or "Urban," depending on the type of project.
4) Project Details: Enter the details of the project including the title, description, date, and project purpose.
5) Project Features: Select any features relevant to the project by toggling the options.
6) Finalise Project: View the project in the dashboard, where you can also edit details or manage user access.

Project Purposes

1) Practice: Use drones to enhance piloting skills and gain confidence in handling equipment.
2) Research: Employ drones for exploring innovative ideas or gathering useful information.
3) Training: Offer opportunities to learn about drone operation and develop skills for various uses.
4) Other: Cater to unique requirements, allowing flexibility and creativity in drone applications.
